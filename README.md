## Run Dev PWA

En la carpeta del proyecto ejecutar los siguientes comandos:

$ npm install http-server -g

$ npm install

$ ng build --prod

$ http-server -p 8080 -c-1 dist/angular-pwa/


En Chrome acceder a: http://localhost:8080/
